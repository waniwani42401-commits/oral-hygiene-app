口腔衛生学 暗記アプリ v9（容量修復版）

GitHubのリポジトリ直下で次の3ファイルを置き換え・追加してください。
- index.html（置き換え）
- sw.js（置き換え）
- update.html（置き換えまたは追加）

更新後、Safariで /oral-hygiene-app/update.html を開き「更新を開始」を押します。
初回起動時に「保存容量を修復します」と表示されたら、ボタンを押してください。
修復前バックアップが自動で保存され、追加画像が学習記録から分離されます。

v9の主な変更
- 追加画像を1枚ずつIndexedDBの別ストアへ保存
- 学習記録を保存するたびに全画像を書き直さない
- 旧形式のBase64画像を自動軽量化・分離
- 不要画像の削除（ガベージコレクション）
- Cache Storageへアプリ本体を複製しない
- 全データ書き出し時には画像をJSONへ再内蔵し、復元可能

口腔衛生学 一問一答 暗記アプリ（全491問）

【主な機能】
・解答を見るまで非表示
・「わかった／分からなかった」の自動保存
・未学習、分からなかった問題、わかった問題だけを絞り込み
・分野別、検索、番号順、シャッフル
・図表は初期非表示。必要な問題だけ「図・表を表示」を押して閲覧し、タップで拡大
・学習データのJSON書き出し／読み込み
・ライト／ダーク表示
・iPhone対応、ホーム画面追加、オフライン対応

【GitHub Pages】
ZIPを展開し、フォルダそのものではなく、中にある全ファイル・全フォルダをリポジトリのルートへアップロードしてください。
特に assets/figures フォルダを省略すると画像が表示されません。
Settings → Pages → Deploy from a branch → main / (root) を選択します。
公開URLをSafariで開き、共有 → ホーム画面に追加、でアプリとして使えます。

【注意】
Service Workerによる完全オフライン動作には、GitHub PagesなどHTTPS上での公開が必要です。
ローカルでindex.htmlを直接開いても学習機能は動きますが、ホーム画面追加・Service Workerは制限されます。


【画像が表示されない場合】
GitHub上で assets → figures → figure-01.jpg が実際に開けるか確認してください。
index.html と assets フォルダは同じ階層に置きます。
古いService Workerが残る場合はSafariのサイトデータを削除するか、ページを再読み込みしてください。

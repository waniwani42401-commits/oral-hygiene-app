口腔衛生学 暗記アプリ v7（大容量保存対応）

変更点
- 学習記録・編集問題・添付画像の保存先を localStorage から IndexedDB に変更
- v6以前の保存データを初回起動時に自動移行
- 端末から追加する画像を自動で軽量化
- 内蔵図表24枚、問題編集、番号移動などは維持

GitHub更新方法
1. 既存リポジトリの一番上の階層で Add file → Upload files
2. このフォルダの index.html と sw.js をアップロードして同名ファイルを置き換える
3. Commit changes
4. 数分後にSafariで公開URLを再読み込み

重要
更新前に、現在開いているアプリから最新の学習データJSONを書き出してください。
容量警告が出た後の未保存変更は、画面を閉じると失われる可能性があります。

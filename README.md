# 口腔衛生学 暗記アプリ PWA v3

## 今回の追加機能
- 図・表は最初は閉じた状態です。
- 「図・表を表示／閉じる」ボタンで開閉できます。
- 前後の問題へ移動すると図・表は自動で閉じます。
- 画面下部の「前の問題／次の問題」は常に表示されます。
- 1〜491の問題番号を入力して直接移動できます。Enterキーでも移動できます。
- 現在の絞り込みに対象問題がない場合は、自動的に絞り込みを解除して移動します。

## GitHub Pagesで既存アプリを更新する場合
画像フォルダは変更していないため、まず次の4ファイルをリポジトリ直下で上書きしてください。
- index.html
- app.js
- styles.css
- sw.js

manifest.webmanifestとiconsは変更しなくても動きますが、完全版では同梱ファイルをすべて使用できます。
更新後に旧版が残る場合は、Safariを再読み込みし、ホーム画面の旧アプリを一度削除してから追加し直してください。

# 口腔衛生学 暗記アプリ v5 差し替え用

## 追加・復元した機能
- 問題の編集
- 問題の削除と復元
- 問題の追加・複製
- 問題管理画面
- 問題一覧画面（問題をタップするとその場で解答が開く）
- 図・表の開閉
- 問題番号による直接移動
- 画面下部に前・次・番号移動を固定

## GitHubで差し替えるファイル
次の4ファイルをリポジトリ直下へ上書きしてください。
- index.html
- app.js
- styles.css
- sw.js

## 絶対に消さないファイル・フォルダ
- base-questions.js（既存491問）
- assets/
- icons/
- manifest.webmanifest
- .nojekyll

`base-questions.js` がリポジトリ直下にない場合は、以前の編集可能版に入っていた同名ファイルを戻してください。

## 更新後に旧画面が残る場合
1. GitHub Pagesの公開を1〜3分待つ
2. Safariで再読み込み
3. ホーム画面版が古い場合は一度削除
4. Safariで公開URLを開き直して「ホーム画面に追加」

学習記録・編集内容は旧版のlocalStorageから自動移行を試みます。念のため、更新前にバックアップを書き出してください。
